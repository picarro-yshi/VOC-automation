VOC-automation
Calibration, experiment, calculation, graphic user interface

Python 3.9, PyQt6 Tested on Windows, Mac and Linux

## Requirements
Many libraries need to be installed before code is able to run.
See /2022.3.8ui2/t5module.py for a list of installed library and run this script to see if requirements are met.

## How to run
- Graphic User Interface: in the folder '2022.3.8ui2', run
t8ui.py

- Fitting the spectrum: in the folder '2022.6.3jupyter2', run
jupy.py


## Features
- Automatic run of both gas tank and droplet calibration experiments
- Data analysis for calibration experiments
- Control multiple instrument: MFC, scale, Analysis
