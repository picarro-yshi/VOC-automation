VOC-automation
Calibration, experiment, calculation, graphic user interface

Python 3.9, PyQt6 Tested on Windows, Mac and Linux

## How to run
- Graphic User Interface: in the folder '2022.1.11ui', run
t3ui.py

- Fitting the spectrum: in the folder '2022.1.4jupyter', run
t5jup.py


## Requirements
Many libraries need to be installed before code is able to run. Also need data file.
See 2022.1.11ui/t5module.py for a list of installed library


## Features
- Automation on equipment include: analyzer, Alicat for flow control, scale for weights
- For flow control experiments, time is saved in the folder and will be automatically loaded.
- Find ports for all connected equipment
